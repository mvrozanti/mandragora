gource — mandragora repo

Renders gource over the mandragora git history into an MP4 you can play inline. Picks the desktop workstation when reachable, falls back to a smaller render on the VPS otherwise. Identical params are cached.

Each render is identified by a hash of (repo HEAD at date_to, date window, length, resolution). The MP4 is cached forever under that hash; the same params return the same file instantly. The default-params video is pre-rendered nightly on the desktop, so a fresh page-load is already warm.

When the desktop is reachable over tailscale it does the work (RTX 5070 Ti, ffmpeg with NVENC). When it's offline, the VPS renders locally under hard caps: ≤120s length, ≤1280px on either side.

Source: mvrozanti/mandragora — compose stack at nix/hosts/mandragora-vps/compose/gource, desktop renderer at nix/modules/system/services/gource-renderer.nix.
